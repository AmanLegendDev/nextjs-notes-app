# Notes App

A simple Notes application built using **Next.js** and **MongoDB**.  
This project demonstrates basic CRUD functionality with API integration and database connectivity.

## Features
- Create a new note
- View all notes
- Delete a note
- REST API using Next.js App Router
- MongoDB integration with Mongoose

## Tech Stack
- Next.js (App Router)
- React
- MongoDB
- Mongoose
- JavaScript

## Project Structure
app/ ├─ api/notes/route.js   # API routes ├─ page.js              # Frontend UI ├─ layout.js lib/ └─ db.js                # Database connection models/ └─ note.js              # Mongoose schema
